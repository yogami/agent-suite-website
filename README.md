# AgentOps Suite - Marketing Website

Marketing landing page for the AgentOps Suite - a collection of 5 enterprise-grade multi-agent infrastructure services.

## Products Featured
1. **Semantic Aligner** - Translation middleware for agent communication
2. **Trust Verifier** - DID/VC-based identity verification
3. **Deadline Enforcer** - SLA monitoring and enforcement
4. **Capability Broker** - Agent service discovery
5. **Fairness Auditor** - Bias and compliance detection

## Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS 3
- TypeScript
- Static Export for Railway

## Development
```bash
npm install
npm run dev
```

## Deployment
Deployed on Railway via GitHub integration.
